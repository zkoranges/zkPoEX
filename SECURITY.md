# Security Policy

This project is provided as-is and is not actively maintained.

## Reporting Vulnerabilities in zkpoex

This tool is archived. No security patches will be issued.

## Reporting Vulnerabilities Discovered Using zkpoex

This tool is for authorized security research only. If you discover a vulnerability in a third-party protocol:

1. Do NOT contact the zkpoex maintainers
2. Follow the affected protocol's responsible disclosure policy
3. If no policy exists, contact the protocol team directly before any public disclosure

The zkpoex contributors have no involvement in vulnerabilities you discover and cannot mediate disputes.
